# 降级为长轮询

> [!TIP]
> 该功能为高级版功能，如有需要请联系交流群（183026419）管理员

在部分特殊网络环境（如政务、涉密项目）中，网关层因安全管控需要部署了网闸。此类网闸设备通常不允许长连接存在。

为解决此问题，可借助中国版提供的强制降级能力，将底层通信机制由长连接自动降级为长轮询。该过程对上层业务完全透明，无需业务侧做任何改造，即可适配网闸环境的网络约束。

## 开启功能

```json
"editorConfig": {
    "customization": {
        // 将Websocket强制降级为长轮询，true 开启 false 关闭
        "polling": true,
    },
}
```
