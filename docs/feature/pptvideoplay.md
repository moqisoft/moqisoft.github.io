# PPT视频播放

> [!TIP]  
> 该功能为高级版功能，如有需要请联系交流群（183026419）管理员

针对 OnlyOffice 社区版 PPT 不支持播放视频，中国版做了能力增强，独家支持视频播放。

用户的 PPT 文件包含的视频，可以直接在编辑器中预览和播放，无需下载或使用外部播放器，提升了 PPT 的表现力和交互性。

**特性**：
- 支持 PPT 中插入的视频，直接在中国版中预览和播放视频
- 使用系统自带的 video 能力做解码与播放，UI 自定义实现，可播放、暂停、拖动时间轴、音量调节、一键静音等
