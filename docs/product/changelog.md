# 更新日志

中国版 与 onlyoffice documentserver 社区版尽量保持同步，一般会在官方版本发布后 1-2 周发布中国版对应版本，某些小版本可能会跳过。如遇某版本中加入功能较复杂，会相应延后发布时间。

新版本默认会保留所有旧版本的特性，除非特意注明某特性被移除。


## v9.3.1

时间: 2026.03.10

- 同步升级为官方最新 9.3.1 版本
- 修复 9.3.0 版本中移动端编辑状态下图标缺失问题

## v9.3.0

时间: 2026.03.02

- 同步升级为官方最新 9.3.0 版本
- 新增动态切换权限 `Api.changePermissions` [动态权限切换](../feature/changepermissions.md)
- 提供对东南亚文字的本地化支持，包括越南语（拉丁字母）、泰语、缅甸语、老挝语、高棉语（柬埔寨）、菲律宾语、以及印尼语/马来语（文莱/新加坡/马来西亚）等，可正常输入、显示和编辑这些文字
- 优化 postgresql 数据库默认配置，防止极端场景下连接不释放
- 修复 example 上传 vsdx 格式提示不支持问题
- 修复与 nextcloud 集成时未提供权限字段，默认权限不可拷贝问题
