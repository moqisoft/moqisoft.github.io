# documentserver 中国版

基于社区版最新源码编译，增加以下功能：

- 移除人数限制
- 移动端可编辑
- 添加公文办公场景常用中文字体
- example 界面全汉化，默认语言为简体中文
- 附赠大量中文配置文档
- 可自定义 logo
- 可自定义品牌
- 可使用连接器及 Automation API
- Websocket 支持强制降级为长轮询

更多功能不断加入中，敬请期待。部分高级功能见 [高级版](./vip.md)

> 该软件仅供学习交流，请勿从事违法活动，商用请支持正版。
