# documentserver 中国版

基于社区版最新源码编译，并结合国情做部分功能增强： 

- 移除人数限制
- 移动端完整编辑能力
- 添加中文办公场景常用字体
- 可使用连接器从外部操作文档
- Websocket 支持降级为长轮询
- 防截图水印，可防止资料泄露   
- 剪切板区分内部外部，可禁止粘贴到外部   
- 独家支持本地字体加速开档、迷你工具栏、ppt视频播放等     
- welcome 及 example 界面全汉化，默认语言为简体中文，对新手非常友好    

更多功能不断加入中，详细请关注[更新日志](./changelog.md)。部分高级功能见 [版本比较](../product/compare.md) 页面
